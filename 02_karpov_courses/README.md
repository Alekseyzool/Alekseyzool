# Проекты Karpov.Courses

| Ноутбук | Цель проекта | Методы и подходы | Стек библиотек |
| --- | --- | --- | --- |
| `AA_AB_karpov_simulator.ipynb` | Проверка новой рекомендательной ленты через A/A и A/B тесты CTR. | Выгрузка событий из ClickHouse, проверка честности сплитования повторными Welch t-тестами, анализ A/B c Welch t-test и Mann–Whitney, сглаживание CTR и пуассоновский бутстрап, визуализация распределений. | `pandas`, `numpy`, `scipy`, `matplotlib`, `seaborn`, `clickhouse_connect`, `swifter` |
| `Final_project_Karpov_3v (2).ipynb` | Финальный проект: оценка повышения цены премиума в дейтинговом приложении. | Очистка CSV и приведение типов, объединение контролей, расчёт метрик (CR, ARPU, ARPPU, Trial share), статистические критерии и бутстрап для uplift, анализ мощности и коррекция множественных проверок. | `pandas`, `numpy`, `scipy`, `statsmodels`, `seaborn`, `matplotlib` |
| `Orbit_prediction.ipynb` | Почасовой прогноз CTR ленты и оценка эффектов от изменений. | Сбор временного ряда из ClickHouse, EDA тепловыми картами, обучение DLT-моделей Orbit (MAP/MCMC), прогноз на будущее, бэктесты Expanding/Rolling, подбор параметров, байесовская диагностика ArviZ, оценка вмешательства через CausalImpact. | `pandas`, `numpy`, `orbit`, `clickhouse_connect`, `matplotlib`, `seaborn`, `scipy`, `arviz`, `causalimpact`, `pycausalimpact`, `swifter` |
| `Predicting_server_loads.ipynb` | Прогноз активности пользователей для планирования серверных мощностей. | Формирование почасовых метрик из ClickHouse, выбор частоты и горизонта, построение Seasonal Naive, Orbit KTRLite и DLT, CatBoost регрессии, скользящее бэктестирование, сравнение по WMAPE/sMAPE и интерпретация компонент. | `pandas`, `numpy`, `orbit`, `catboost`, `clickhouse_connect` |
| `ab_pricing_experiment_analysis.ipynb` | Анализ A/B‑теста изменения цены премиума. | Загрузка сырых CSV, чистка пропусков и типов, формирование 14‑дневного окна, проверка эквивалентности контролей, расчёт метрик (Users, Payers, CR, ARPU, ARPPU), критерии χ² и Mann–Whitney, визуализация и бизнес‑выводы. | `pandas`, `numpy`, `matplotlib`, `seaborn`, `scipy`, `os` |
| `ab_test_analysis.ipynb` | Исследование эксперимента по цене премиума в дейтинговом приложении. | Проверка качества данных, агрегирование метрик (conversion, ARPU), пропорциональный z‑тест и Welch t-test, анализ сегментов по странам, графический разбор распределений и выводы. | `pandas`, `numpy`, `matplotlib`, `scipy`, `statsmodels`, `os` |
| `ab_test_project_3_notebook.ipynb` | Вариант 3 проекта: оценка теста цены с двумя контролями. | Гибкая загрузка CSV с autodetect кодировок, очистка и унификация users/transactions, сводные метрики, анализ доли триала, Z‑тесты для конверсий, Mann–Whitney для выручки, бутстрап uplift, сегментация и экспорт отчётов. | `pandas`, `numpy`, `scipy`, `pathlib` |
| `churn_customer_bank.ipynb` | Прогноз оттока клиентов банка (F1 ≥ 0.59). | Предобработка и кодирование признаков, разделение train/valid/test, масштабирование признаков, обучение Logistic Regression, Decision Tree и RandomForest, подбор порогов и сравнение по F1 и AUC‑ROC. | `pandas`, `numpy`, `scikit-learn`, `warnings` |
